# Objetivos

## Objetivo general

Desarrollar una solución académica integral sobre el sistema de transporte masivo local tipo Ciudad Móvil, reutilizando la red previamente construida para implementar un modelo de aprendizaje supervisado orientado a la predicción del tiempo estimado de viaje y un modelo de aprendizaje no supervisado orientado al descubrimiento de patrones operativos en los trayectos.

## Objetivos específicos

1. Construir un dataset sintético coherente con la red de transporte previamente diseñada, incluyendo variables operativas y contextuales relevantes para el análisis.
2. Formular el problema de predicción del tiempo estimado de viaje como una tarea de regresión supervisada.
3. Entrenar y comparar al menos dos modelos de regresión, específicamente Linear Regression y Random Forest Regressor.
4. Evaluar el desempeño de los modelos supervisados mediante métricas cuantitativas como MAE, MSE, RMSE y R².
5. Seleccionar y justificar el modelo supervisado con mejor comportamiento sobre el conjunto de prueba.
6. Preparar un dataset derivado para aplicar técnicas de aprendizaje no supervisado sobre las características operativas de los trayectos.
7. Implementar un algoritmo de clustering basado en K-Means y analizar distintos valores de k mediante método del codo y silhouette score.
8. Interpretar los clusters obtenidos en lenguaje académico, relacionándolos con perfiles operativos de trayectos dentro del sistema de transporte.
9. Documentar la metodología, las pruebas realizadas, los resultados obtenidos y las conclusiones del proyecto de forma clara, coherente y lista para entrega universitaria.
